# Capability mapping checklist — canonical reference (rev 2026-06-12, codifies the the framework SES capability mapping convention)

## When this applies

APS SES Band 1/2 panel briefs and state-government / board-level equivalents scored against a formal capability framework. The canonical frame is the APS Integrated Leadership System (ILS); organisation-specific values frameworks (APS Values, Secretaries Charter DRIVE, the agency's own values) layer on top.

## Component 1 — Capability groups reference

A short reference list of the applicable groups. APS ILS SES Band 1 has five:

1. Shapes strategic thinking
2. Achieves results
3. Cultivates productive working relationships
4. Exemplifies personal drive and integrity
5. Communicates with influence

If the role uses a different framework, the brief must name it and list its groups.

## Component 2 — Per-question capability mapping

A table per question:

| Capability tested | Where the response covers it | Rating |
|---|---|---|
| [group] | [sentence/segment of the tailored response] | Strong / Adequate / Watch |

**Every cell must carry a rating.** An unrated cell is an Incomplete finding (Medium).

## Component 3 — Overall coverage matrix

Capabilities × questions, at-a-glance — where the panel gets strong evidence and where coverage thins. Every framework group appears as a row; every planned question as a column.

## Component 4 — Watch pivot plan

For every capability rated **Watch** anywhere in the mapping:

- A named pivot story with the underlying STAR ID (e.g. "pivot to STAR 5.4 — vendor model redesign") the candidate uses if the panel probes that capability in a follow-up.
- The **trigger phrases** to listen for (the panel wordings that signal the probe is coming).

A Watch without a named pivot story + trigger phrases is a **High** finding.

## Component 5 — Values and behaviours coverage summary

A cross-frame matrix showing which agency values, conduct codes (e.g. DRIVE) and APS Values each question touches. Required whenever the organisation publishes a values framework.

## Component 6 — "One thing to remember on the day"

The cumulative-impression reminder, verbatim or equivalent:

> "The panel will not score the four questions in isolation — they will score the cumulative impression across all four. Do not try to demonstrate every capability in every answer. Trust the matrix."

## Supplementary checks

- **Question-to-STAR mapping callout** at the top of the brief's Section 5 once confirmed questions arrive (~24h out), mirroring Section A of the addendum.
- **≤30-minute panels:** questions back to the panel constrained to 2 primary + 3 backup, with explicit swap guidance.

## Severity

| Severity | Trigger |
|---|---|
| **High** | Any of the six components missing; any Watch without a pivot story |
| **Medium** | Component present but incomplete (unrated cells, matrix missing a group, pivot without trigger phrases) |
| **Low** | Style/placement issues (e.g. reminder present but buried) |
