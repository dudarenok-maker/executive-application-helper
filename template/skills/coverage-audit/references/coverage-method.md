# Coverage analysis method — canonical reference (rev 2026-06-12, codifies the framework STEP 2)

## What gets mapped

Map the draft against, where provided:

- Candidate Information Pack
- Position Description
- Advertisement
- Recruiter-specific requests
- Selection criteria
- Any "About You", "What We Are Looking For", "Key Accountabilities", "Key Challenges", "Capabilities" or similar section

## Status definitions

| Status | Meaning |
|---|---|
| **Addressed** | The draft carries a direct, evidenced response — specific passage, anchored by a bank entry, calibrated to the requirement's seniority and scope |
| **Partially Addressed** | Touched but under-evidenced: generic claim without proof point, adjacent example rather than direct, or covers one half of a two-part requirement |
| **Not Addressed** | No draft passage maps to the requirement |

## Table format (one per source document)

| Section / requirement | Status | How covered (entry IDs) | Gap / risk |
|---|---|---|---|
| [verbatim or tight paraphrase] | Addressed / Partial / Not | [draft passage gist + A/B entry IDs] | [one line, or —] |

Entry IDs are cited for traceability — they never appear in the reader-facing draft itself.

## After the tables

1. **Risk list** — important requirements not strongly evidenced; areas where further examples would improve the application; draft wording that needs strengthening or sharpening.
2. **Bank-gap candidates** — any Not/Partially Addressed item that traces to *missing bank evidence* (not a drafting choice) is named for the STEP 4 maintenance report as a capture candidate. Do not park these silently.

## Judgement notes

- Prioritise relevance over completeness: a deliberately unaddressed minor requirement in a tight word band is a drafting choice, not a failure — note it as such.
- A requirement addressed only by the resume (not the drafted document) is **Partially Addressed** from the document's standpoint; note where the resume carries it.
- Where the same evidence carries multiple requirements, check the draft doesn't repeat the same point in different words — flag if it does.
